# ![My Skills](https://skillicons.dev/icons?i=unity) Horror Game

- Game
  - Type: Horror
  - Goal: Find necessary items, figure out puzzles and escape.
  - Place: Home
  - Platform: PC
  - Control: WASD keys, arrow keys, E key (interact with items), tab key (menu)


- Technologies
   - Unity
   - ProBuilder
   - Blender
